# 02_findings_and_actions.md

## Objective / 目的

- This document summarizes audit findings and proposes corrective and preventive actions for continuous improvement of the ISMS framework. 
	本書は、監査で確認された指摘事項を整理し、ISMS体制の継続的改善を目的とした是正・予防措置を提案する。

---
## Findings and Actions / 指摘事項および対応策

| ID   | Category / 区分                | Finding / 指摘内容                                    | Root Cause / 原因                                                                                | Corrective Action / 是正措置                                                               | Preventive Action / 予防措置                                                                                            | Status / 状況       |
| ---- | ---------------------------- | ------------------------------------------------- | ---------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- | ----------------- |
| F-01 | Documentation / 文書管理         | Outdated access control procedure / アクセス制御手順書の未更新 | No ownership defined for document update process / 文書改訂責任者が不明確                                 | Update the document and assign an owner responsible for revisions / 文書を更新し、改訂責任者を明確にする | Implement a revision control system for policy documents / 文書改訂管理システムを導入する                                          | In progress / 進行中 |
| F-02 | Incident Response / インシデント対応 | Missing post-incident review / 事後報告書作成漏れ          | Workflow lacked a defined closure step; responsible role not defined / ワークフローに終了手順と責任者の定義がなかった | Conduct a post-incident review and record lessons learned / 事後レビューを実施し、教訓を記録する         | Add a closure step and assign responsibility within the incident response checklist / インシデント対応チェックリストに完了工程と責任者を追加する | Completed / 完了    |


## Follow-up Audit Preparation / フォローアップ監査準備
- After implementing the corrective and preventive actions, a follow-up audit will be conducted to verify the effectiveness of each measure.  
- This process ensures that all identified nonconformities have been properly addressed and that improvements are sustained within the management system.
	是正および予防処置の実施後、それぞれの対応が効果的に実行されたかを確認するため、フォローアップ監査を実施します。  
	これにより、指摘事項が適切に解消され、改善が継続的に維持されていることを確認します。
